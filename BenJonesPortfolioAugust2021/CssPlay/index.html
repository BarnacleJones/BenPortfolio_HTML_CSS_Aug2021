    Images and graphics

    To insert an image - <img src="url or filepath" alt="alt text" "width=300" "height = 300">

    There are 4 main image formats at the time of writing

  • Gif - limited colours (256) good at compressing large areas of single colour - pixelates photographs
  • SVG - good for icons, logos, a vector file (programming language) so can scale quite large
  • JPG - great for photos, just specify size, can be compressed
  • PNG - images that need transparency

    • Responsive Images

    To use responsive images (images that will scale depending on internet connection, type of device, etc you will need to use CSS

    This page will go into further detail. Very good practice!

    Media

    To insert audio in your page - use the <audio> tag

    Specify the source as well, and use 'controls' so that you can play and pause the audio -
    <audio controls src="audio/1 - The River.mp3"></audio>

    To insert a video use the <video> tag and you will need to include the src="" and type="" attributes

    If you are inserting a youtube video you can use the 'embed' link (click share video for this) and simply copy that into your code

    Identifying content

    Supporting languages:

    You are able to specify the language you are using

    In the head section just put <html lang="en-US" or "en-GB"> to specify which language you are using.

    You can also specify (in the head section) which direction the text reads - LTR or RTL

    <dir ltr >

    Div vs span

    The <div> tag is a block level element (starts a new line and takes up full width)

    The <span> tag is an inline element (in line means just marks up a part of text, not whole line)

    Please refer to the W3Schools page for div and span information!

    These are easily styled by CSS. DIV is used as a container for HTML elements which you can then style

    Tables

    To make a table use the following code which will produce the example below:

    <table>
    <tr>
    <th>Month</th>
    <th>Savings</th>
    </tr>
    <tr>
    <td>January</td>
    <td>$100</td>
    </tr>
    <tr>
    <td>February</td>
    <td>$150</td>
    </tr>
    </table>


    <table>- will initialise the table.
    <tr>- is for defining a new row
    <th> - defines the table header.
    <td> - defines the table data.
